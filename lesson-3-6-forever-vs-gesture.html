Forever loop vs Gesture event

Drag the slider to tilt Device A left or right. Press Shake. Watch how each block type responds differently.
Tilt Device A
Forever loop
Reads the sensor every 500ms — constantly, no matter what
forever {
  send X value
  wait 500ms
}
A reading arrives every 500ms →
Sending:
0
—
Gesture event
Fires once when movement happens — silent otherwise
on tilt left {
  send "LEFT"
}
on tilt right {
  send "RIGHT"
}
Only fires on movement →
Sending:
nothing
—
Forever loop — like a CCTV camera recording all the time. Every reading is sent, even when nothing changes. Good for sending live sensor values continuously.

Gesture event — like a motion sensor. Silent until something moves, then fires once. Good for detecting a specific movement and sending a direction command.

Important: Delete the forever loop before adding gesture blocks. Running both at the same time floods the radio with numbers and drowns out the gesture signal.
